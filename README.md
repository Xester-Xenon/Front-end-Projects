# Packers and Movers — Static Website

This is a simple and clean **Packers and Movers** website designed using HTML, CSS. The main purpose of this project was to practice front-end web development and build a professional looking website layout for a logistics and shifting service business.

---

## Features

- Fully static website — no backend required
- Professional homepage design
- Dedicated sections/pages like:
  - Services
  - About Us
  - Contact
- Responsive layout (works on different screen sizes)
- Simple and easy-to-navigate UI
- Attractive use of images/icons to represent services

---

## Technologies Used

| Technology | Purpose |
|-----------|---------|
| HTML5 | Page structure |
| CSS3 | Styling and layout |
| Bootstrap | Responsiveness and components |
| Icons/Images | Improved visuals |
